# Images Folder

Place all website images in this folder.

## Recommended Images

| Filename              | Used In              | Description                         |
|-----------------------|----------------------|-------------------------------------|
| `logo.png`            | All pages            | OTM logo (transparent background)   |
| `hero-bg.jpg`         | index.html           | Hero section background             |
| `about-banner.jpg`    | about.html           | About page feature image            |
| `event-default.jpg`   | events.html          | Default event card image            |
| `news-default.jpg`    | news.html            | Default news thumbnail              |
| `avatar-default.png`  | Admin panel          | Default user avatar                 |
| `team-1.jpg`          | about.html           | Team member photo                   |
| `team-2.jpg`          | about.html           | Team member photo                   |
| `team-3.jpg`          | about.html           | Team member photo                   |
| `team-4.jpg`          | about.html           | Team member photo                   |

## Usage in HTML

```html
<img src="src/images/logo.png" alt="OTM Logo" />
```

## Notes
- Recommended formats: `.jpg`, `.png`, `.webp`
- Keep images compressed for fast load times
- Logo should be at least 200x200px
- Banner images: 1200x600px recommended
